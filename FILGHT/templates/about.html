Flight Path AI Project

Flight Path Optimization using QAOA

About The Project

This project is a comprehensive platform for optimizing and analyzing flight routes using AI and advanced algorithms. It integrates airport data, flight information, and machine learning models to provide efficient flight path predictions and operational insights.

Key Features:

  • Flight Path Optimization: Uses QAOA and other optimizers for route planning.
  • AI Model Integration: Predicts flight angles and paths using trained Keras models.
  • Interactive Visualization: Map-based visualization of flight paths and airport locations.
  • Gemini AI Chatbot: Integrated AI for intelligent chatbot interactions.
